Teacher Reflection

This website, LearnersHub, was designed using a student-centered approach where learners take an active role in their learning process. The aim was to create an online learning environment that allows students to access programming notes, watch tutorial videos, and complete quizzes independently.

Teaching Approach Used

I used the student-centered learning approach because it encourages independent learning, problem-solving, and active participation. Learners are able to study at their own pace and revisit lessons whenever needed.

Learning Materials Included

The website includes programming notes, Object-Oriented Programming (OOP) resources, video tutorials, and quizzes. These materials support different learning styles such as visual, reading, and practical learning.

Accessibility and Inclusivity

The website was designed with simple navigation, clear colors, readable text, and mobile-friendly layout to ensure accessibility for all learners. Students can easily use the platform even with limited technical skills.

Challenges Faced

During development, I faced challenges with JavaScript quiz functionality, page layout design, and publishing the website online. These challenges were solved through practice, online research, and testing different solutions.

Future Improvements

In future, I would improve the website by adding downloadable PDF notes, discussion forums, live tutoring support, and more interactive assessments to improve learner engagement.

Conclusion

This project helped me combine teaching methods with technology to support effective learning. It showed how educational websites can improve access to learning resources and promote independent study among students.
